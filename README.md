### Prerequisites

Make sure you have the following installed on your machine:

- Node.js
- npm or yarn
- MongoDB (for storing chat data)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/chat-app.git

2. Install dependencies:

   ```bash
   git install

3. Run the App:

   ```bash
   npm run dev
